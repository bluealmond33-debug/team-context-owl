# Inbox

Unsorted material lands here. Log every intake in `INTAKE_INDEX.md` first. Keep originals in `_raw/`, and when filing, move items to their destination folder with `git mv` instead of deleting.

> **When is an item "done"?** Only after all four steps in `AGENTS.md` → "Inbox processing" are complete **in one commit**: digest in 01–07, original moved to the destination folder's `_raw/`, origin link pointing at the final location, and the item removed from here (ledger row set to `classified`).
>
> ⚠️ **This folder does not empty itself.** If anything besides `README.md`, `INTAKE_INDEX.md`, and an empty `_raw/` remains, processing is still in progress.
